2. Sintaxis Básica de Java

Fundamentos del lenguaje: variables, operadores y estructuras de control

Variables y Tipos de Datos

Una variable es un espacio en memoria donde almacenamos datos. En Java, cada variable tiene un tipo que define qué clase de datos puede contener y qué operaciones puedes realizar con ella.

Java es un lenguaje fuertemente tipado, lo que significa que debes declarar el tipo de cada variable antes de usarla, y una vez declarada, no puedes cambiar su tipo.

Tipos Primitivos

Java tiene 8 tipos primitivos que almacenan valores simples:

  • byte: Números enteros de -128 a 127 (8 bits)
  • short: Números enteros de -32,768 a 32,767 (16 bits)
  • int: Números enteros de -2,147,483,648 a 2,147,483,647 (32 bits) - el más usado
  • long: Números enteros muy grandes (64 bits) - añade 'L' al final
  • float: Números decimales (32 bits) - añade 'f' al final
  • double: Números decimales de mayor precisión (64 bits) - el más usado para decimales
  • char: Un solo carácter Unicode - usa comillas simples 'A'
  • boolean: true o false
// Declaración e inicialización de variables
int edad = 25;                    // Entero
double precio = 19.99;            // Decimal
char inicial = 'A';               // Carácter
boolean activo = true;            // Booleano
String nombre = "Ana";            // String (tipo referencia)

// Puedes declarar sin inicializar
int contador;
contador = 0;  // Inicialización posterior

// Múltiples variables del mismo tipo
int x = 5, y = 10, z = 15;

Convenciones de Nombres

  • Usa camelCase: primera letra minúscula, siguientes palabras con mayúscula inicial
  • Nombres descriptivos: edadUsuario mejor que e
  • Constantes en MAYÚSCULAS: final int MAX_INTENTOS = 3;
  • No uses palabras reservadas: int, class, public, etc.

Operadores

Los operadores son símbolos que realizan operaciones sobre variables y valores.

Operadores Aritméticos

int a = 10, b = 3;

int suma = a + b;        // 13
int resta = a - b;       // 7
int multiplicacion = a * b;  // 30
int division = a / b;    // 3 (división entera)
int modulo = a % b;      // 1 (resto de la división)

// Incremento y decremento
a++;  // a = a + 1;  (ahora a es 11)
b--;  // b = b - 1;  (ahora b es 2)

Operadores de Comparación

Retornan un valor booleano (true/false):

int x = 5, y = 10;

boolean igual = (x == y);        // false
boolean diferente = (x != y);    // true
boolean mayor = (x > y);         // false
boolean menor = (x < y);         // true
boolean mayorIgual = (x >= 5);   // true
boolean menorIgual = (y <= 10);  // true

Operadores Lógicos

boolean a = true, b = false;

boolean and = a && b;    // false (ambos deben ser true)
boolean or = a || b;     // true (al menos uno debe ser true)
boolean not = !a;        // false (invierte el valor)

// Ejemplo práctico
int edad = 20;
boolean mayorEdad = edad >= 18 && edad < 65;  // true

Estructuras Condicionales

if - else if - else

Permite ejecutar código solo si se cumple una condición: 

int nota = 85;

if (nota >= 90) {
    System.out.println("Excelente");
} else if (nota >= 70) {
    System.out.println("Bien");
} else if (nota >= 50) {
    System.out.println("Aprobado");
} else {
    System.out.println("Suspendido");
}

switch

Útil cuando comparas una variable con múltiples valores exactos: 

int dia = 3;
String nombreDia;

switch (dia) {
    case 1:
        nombreDia = "Lunes";
        break;
    case 2:
        nombreDia = "Martes";
        break;
    case 3:
        nombreDia = "Miércoles";
        break;
    default:
        nombreDia = "Otro día";
        break;
}

System.out.println(nombreDia);  // Miércoles

Operador Ternario

Forma compacta de if-else para asignaciones simples: 

int edad = 20;
String mensaje = (edad >= 18) ? "Mayor de edad" : "Menor de edad";
// Si edad >= 18 es true, mensaje = "Mayor de edad"
// Si es false, mensaje = "Menor de edad"

Bucles (Loops)

Bucle for

Usa for cuando sabes cuántas veces quieres repetir: 

// Imprimir números del 1 al 10
for (int i = 1; i <= 10; i++) {
    System.out.println(i);
}

// Estructura: for (inicialización; condición; incremento)
// 1. Se ejecuta la inicialización (int i = 1)
// 2. Se verifica la condición (i <= 10)
// 3. Si es true, ejecuta el código dentro
// 4. Ejecuta el incremento (i++)
// 5. Vuelve al paso 2

Bucle while

Usa while cuando no sabes cuántas veces se repetirá: 

int contador = 0;
while (contador < 5) {
    System.out.println("Contador: " + contador);
    contador++;
}

// Se ejecuta mientras la condición sea true
// Si la condición es false desde el inicio, nunca se ejecuta

Bucle do-while

Similar a while, pero se ejecuta al menos una vez: 

int numero;
do {
    numero = (int)(Math.random() * 10);
    System.out.println("Número: " + numero);
} while (numero != 5);

// Se ejecuta el código primero
// Luego verifica la condición
// Garantiza al menos una ejecución

Bucle for-each

Ideal para recorrer arrays o colecciones: 

String[] nombres = {"Ana", "Luis", "María"};

for (String nombre : nombres) {
    System.out.println(nombre);
}

// Lee: "por cada nombre en nombres"
// No necesitas índices

Break y Continue

// break: sale completamente del bucle
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;  // Sale cuando i es 5
    }
    System.out.println(i);  // Imprime: 1, 2, 3, 4
}

// continue: salta a la siguiente iteración
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;  // Salta cuando i es 3
    }
    System.out.println(i);  // Imprime: 1, 2, 4, 5
}

Arrays

Un array es una colección de elementos del mismo tipo almacenados en posiciones consecutivas de memoria. Tiene un tamaño fijo que se define al crearlo.

Declaración e Inicialización

// Forma 1: Declarar y crear
int[] numeros = new int[5];  // Array de 5 enteros (todos son 0 por defecto)

// Forma 2: Declarar, crear e inicializar
int[] edades = {18, 25, 30, 22, 28};

// Forma 3: Declaración separada
String[] nombres;
nombres = new String[3];

// Arrays de diferentes tipos
double[] precios = new double[10];
boolean[] estados = {true, false, true};

Acceso a Elementos

Los arrays en Java usan índices que empiezan en 0: 

String[] frutas = {"Manzana", "Pera", "Naranja"};

// Acceder a elementos (lectura)
String primera = frutas[0];   // "Manzana"
String segunda = frutas[1];   // "Pera"

// Modificar elementos
frutas[2] = "Plátano";        // Ahora es {"Manzana", "Pera", "Plátano"}

// Obtener la longitud
int tamaño = frutas.length;   // 3 (¡sin paréntesis!)

Recorrer Arrays

int[] numeros = {10, 20, 30, 40, 50};

// Con for tradicional (cuando necesitas el índice)
for (int i = 0; i < numeros.length; i++) {
    System.out.println("Posición " + i + ": " + numeros[i]);
}

// Con for-each (más simple cuando solo lees)
for (int numero : numeros) {
    System.out.println(numero);
}

// Calcular la suma
int suma = 0;
for (int numero : numeros) {
    suma += numero;
}
System.out.println("Suma: " + suma);  // 150

Arrays Multidimensionales

Arrays de arrays (matrices): 

// Matriz 3x3
int[][] matriz = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Acceder a elementos
int valor = matriz[1][2];  // 6 (fila 1, columna 2)

// Recorrer matriz
for (int i = 0; i < matriz.length; i++) {
    for (int j = 0; j < matriz[i].length; j++) {
        System.out.print(matriz[i][j] + " ");
    }
    System.out.println();
}

Errores Comunes con Arrays

  • ArrayIndexOutOfBoundsException: Intentar acceder a un índice que no existe
  • Recuerda: si un array tiene 5 elementos, los índices van de 0 a 4
  • No puedes cambiar el tamaño de un array después de crearlo
  • length es una propiedad, no un método (sin paréntesis)
