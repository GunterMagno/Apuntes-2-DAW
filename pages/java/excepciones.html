5. Manejo de Excepciones

Gestión de errores en Java

¿Qué son las Excepciones?

Una excepción es un evento que ocurre durante la ejecución de un programa y interrumpe el flujo normal del código. Por ejemplo: dividir por cero, acceder a un archivo que no existe, o intentar usar un índice fuera de rango en un array.

Sin manejo de excepciones: El programa se detiene abruptamente.

Con manejo de excepciones: Puedes capturar el error, mostrar un mensaje apropiado, y continuar la ejecución del programa.

Try-Catch

El bloque try-catch permite intentar ejecutar código que podría fallar:

try {
    int resultado = 10 / 0;  // División por cero
    System.out.println(resultado);
} catch (ArithmeticException e) {
    System.out.println("Error: No se puede dividir por cero");
}
System.out.println("El programa continúa...");

Try-Catch-Finally

El bloque finally siempre se ejecuta, haya o no excepción:

try {
    // Código que puede fallar
    int[] numeros = {1, 2, 3};
    System.out.println(numeros[5]);
} catch (ArrayIndexOutOfBoundsException e) {
    System.out.println("Índice fuera de rango");
} finally {
    System.out.println("Esto siempre se ejecuta");
}

Múltiples Catch

try {
    String texto = null;
    System.out.println(texto.length());
} catch (NullPointerException e) {
    System.out.println("El texto es null");
} catch (Exception e) {
    System.out.println("Otro error: " + e.getMessage());
}

Tipos de Excepciones

Checked Exceptions: Debes manejarlas obligatoriamente (ej: IOException)

Unchecked Exceptions: Opcionales de manejar (ej: NullPointerException)

  • NullPointerException: Intentar usar un objeto que es null
  • ArrayIndexOutOfBoundsException: Índice fuera del array
  • ArithmeticException: Error aritmético (división por cero)
  • NumberFormatException: Convertir String inválido a número
  • FileNotFoundException: Archivo no encontrado

Throws y Throw

throws: Declara que un método puede lanzar una excepción

public void leerArchivo() throws IOException {
    // Código que puede lanzar IOException
}

throw: Lanza manualmente una excepción

public void setEdad(int edad) {
    if (edad < 0) {
        throw new IllegalArgumentException("La edad no puede ser negativa");
    }
    this.edad = edad;
}

Excepciones Personalizadas

public class SaldoInsuficienteException extends Exception {
    public SaldoInsuficienteException(String mensaje) {
        super(mensaje);
    }
}

// Uso
public void retirar(double cantidad) throws SaldoInsuficienteException {
    if (cantidad > saldo) {
        throw new SaldoInsuficienteException("Saldo insuficiente");
    }
    saldo -= cantidad;
}
