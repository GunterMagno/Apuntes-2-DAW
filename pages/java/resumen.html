Resumen Java

Comandos y conceptos esenciales

Comandos Java (CLI)

Comando Parámetros Descripción
javac archivo.java Compila archivos .java a bytecode (.class)
java NombreClase Ejecuta una clase compilada con método main
java -version - Muestra la versión de Java instalada
javac -d directorio archivo.java Compila y guarda .class en directorio especificado
java -cp classpath NombreClase Ejecuta con un classpath específico
jar -cvf archivo.jar archivos Crea un archivo JAR
jar -xvf archivo.jar Extrae contenido de un JAR

Conceptos Clave

Concepto Sintaxis/Ejemplo Descripción
Clase public class MiClase { } Plantilla para crear objetos
Método main public static void main(String[] args) Punto de entrada de la aplicación
Constructor public MiClase() { } Inicializa objetos de la clase
Herencia class Hijo extends Padre Una clase hereda de otra
Interface interface MiInterface { } Contrato de métodos a implementar
ArrayList ArrayList<String> lista = new ArrayList<>(); Lista dinámica de elementos
HashMap HashMap<K, V> map = new HashMap<>(); Estructura clave-valor
Try-Catch try { } catch (Exception e) { } Manejo de excepciones
Stream lista.stream().filter().collect() Procesamiento funcional de colecciones

Métodos Comunes de Colecciones

Método Parámetros Descripción
.add() elemento Añade un elemento a la colección
.remove() elemento/índice Elimina un elemento
.get() índice Obtiene elemento por índice
.size() - Retorna el tamaño de la colección
.clear() - Vacía la colección
.contains() elemento Verifica si contiene el elemento
.isEmpty() - Verifica si está vacía
