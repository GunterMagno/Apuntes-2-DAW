Resumen Node.js

Comandos NPM y módulos esenciales

Comandos NPM/Node

Comando Parámetros/Flags Descripción
npm init -y (salta preguntas) Crea package.json
npm install paquete o vacío Instala paquete o todas las dependencias
npm install paquete --save-dev / -D Instala como dependencia de desarrollo
npm install paquete -g Instala paquete globalmente
npm uninstall paquete Desinstala un paquete
npm update paquete o vacío Actualiza paquete(s)
npm run script Ejecuta script definido en package.json
npm start - Ejecuta script "start"
npm test - Ejecuta script "test"
node archivo.js Ejecuta archivo JavaScript
node -v - Muestra versión de Node.js
npm list -g (globales) Lista paquetes instalados

Módulos Core de Node.js

Módulo Importación Descripción
fs const fs = require('fs') Manejo de archivos (File System)
path const path = require('path') Manipulación de rutas de archivos
http const http = require('http') Crear servidores HTTP
https const https = require('https') Crear servidores HTTPS
events const EventEmitter = require('events') Sistema de eventos
os const os = require('os') Información del sistema operativo
crypto const crypto = require('crypto') Funciones criptográficas
url const url = require('url') Parseo y manipulación de URLs

Métodos del módulo FS

Método Parámetros Descripción
fs.readFile() path, encoding, callback Lee archivo de forma asíncrona
fs.readFileSync() path, encoding Lee archivo de forma síncrona
fs.writeFile() path, data, callback Escribe archivo (sobrescribe)
fs.appendFile() path, data, callback Añade contenido al final del archivo
fs.unlink() path, callback Elimina archivo
fs.mkdir() path, callback Crea directorio
fs.readdir() path, callback Lee contenido de directorio
fs.existsSync() path Verifica si existe archivo/directorio

Async/Await & Promises

Concepto Sintaxis Descripción
Promise new Promise((resolve, reject) => {}) Objeto para operaciones asíncronas
.then() promise.then(data => {}) Ejecuta cuando Promise se resuelve
.catch() promise.catch(err => {}) Maneja errores de Promise
async async function nombre() {} Declara función asíncrona
await const data = await promise Espera resolución de Promise
Promise.all() Promise.all([p1, p2]) Espera múltiples Promises en paralelo
