1. Introducción a Node.js

JavaScript del lado del servidor

¿Qué es Node.js?

Node.js es un entorno de ejecución de JavaScript construido sobre el motor V8 de Chrome. Permite ejecutar JavaScript fuera del navegador, en el servidor.

Características principales:

  • JavaScript en el backend
  • No bloqueante (asíncrono)
  • Basado en eventos
  • NPM (gestor de paquetes)

Event Loop

El Event Loop es el corazón de Node.js. Permite manejar múltiples operaciones sin bloquear el hilo principal.

Funcionamiento:

  1. Ejecuta código síncrono
  2. Registra callbacks de operaciones asíncronas
  3. Cuando termina una operación, ejecuta su callback
  4. Repite el ciclo

Asincronía

Node.js es no bloqueante. Las operaciones I/O (lectura de archivos, peticiones HTTP) se ejecutan en segundo plano.

// Código bloqueante (mal)
const data = fs.readFileSync('archivo.txt');
console.log(data);
console.log('Siguiente línea');

// Código no bloqueante (bien)
fs.readFile('archivo.txt', (err, data) => {
    console.log(data);
});
console.log('Siguiente línea');  // Se ejecuta primero

Node.js vs JavaScript del Navegador

En el navegador:

  • Tienes acceso al DOM (document, window)
  • APIs del navegador (fetch, localStorage)

En Node.js:

  • NO hay DOM
  • Acceso al sistema de archivos
  • Módulos del sistema (fs, http, path)
  • Variables globales: __dirname, __filename, process
