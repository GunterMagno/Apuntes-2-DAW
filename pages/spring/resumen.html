Resumen Spring Boot

Anotaciones y comandos esenciales

Comandos Maven

Comando Parámetros Descripción
mvn spring-boot:run - Ejecuta la aplicación Spring Boot
mvn clean install - Limpia, compila y genera el JAR
mvn clean package - Compila y empaqueta sin instalar
mvn test - Ejecuta los tests
mvn dependency:tree - Muestra árbol de dependencias
java -jar app.jar Ejecuta el JAR generado

Anotaciones Principales

Anotación Uso Descripción
@SpringBootApplication Clase principal Habilita configuración automática y escaneo de componentes
@RestController Clase Controlador REST que retorna JSON directamente
@Controller Clase Controlador MVC que retorna vistas
@Service Clase Marca clase como servicio de lógica de negocio
@Repository Interfaz/Clase Marca clase como repositorio de datos
@Autowired Campo/Constructor Inyección automática de dependencias
@Entity Clase Marca clase como entidad JPA (tabla BD)
@Table Clase Especifica nombre de tabla en BD
@Id Campo Marca campo como clave primaria
@GeneratedValue Campo Generación automática de ID
@Column Campo Configura columna de BD

Anotaciones de Mapping (REST)

Anotación Ejemplo Descripción
@GetMapping @GetMapping("/users") Maneja peticiones GET
@PostMapping @PostMapping("/users") Maneja peticiones POST
@PutMapping @PutMapping("/users/{id}") Maneja peticiones PUT (actualizar)
@DeleteMapping @DeleteMapping("/users/{id}") Maneja peticiones DELETE
@PatchMapping @PatchMapping("/users/{id}") Maneja peticiones PATCH (actualización parcial)
@RequestMapping @RequestMapping("/api") Mapeo genérico de rutas (nivel clase o método)
@PathVariable @PathVariable Long id Captura variable de la URL
@RequestBody @RequestBody User user Recibe objeto JSON del body
@RequestParam @RequestParam String name Captura parámetro de query (?name=valor)

Métodos JPA Repository

Método Ejemplo Descripción
save() repository.save(entity) Guarda o actualiza entidad
findById() repository.findById(id) Busca por ID
findAll() repository.findAll() Obtiene todas las entidades
deleteById() repository.deleteById(id) Elimina por ID
existsById() repository.existsById(id) Verifica si existe por ID
count() repository.count() Cuenta total de registros
