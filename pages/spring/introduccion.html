1. Introducción a Spring Boot

Framework para aplicaciones Java

¿Qué es Spring Boot?

Spring Boot facilita la creación de aplicaciones Spring listas para producción con configuración mínima.

Ventajas:

  • Configuración automática
  • Servidor embebido (Tomcat)
  • Sin XML, solo anotaciones
  • Producción lista

Spring Initializr

Genera proyectos Spring Boot desde https://start.spring.io

Seleccionar:
- Maven/Gradle
- Java version
- Dependencies: Spring Web, Spring Data JPA, H2 Database

Configuración

application.properties

server.port=8080
spring.application.name=MiApp

# Base de datos
spring.datasource.url=jdbc:h2:mem:testdb
spring.datasource.username=sa
spring.datasource.password=

# JPA
spring.jpa.show-sql=true
spring.jpa.hibernate.ddl-auto=update

Clase Principal

@SpringBootApplication
public class DemoApplication {
    public static void main(String[] args) {
        SpringApplication.run(DemoApplication.class, args);
    }
