Box Model - Modelo de Caja CSS

¿Qué es el Box Model?

El Box Model describe cómo se calcula el espacio que ocupa un elemento en una página web. Cada elemento es una caja rectangular compuesta por cuatro áreas:

Componentes del Box Model (de dentro hacia fuera):

  • Content: El contenido real (texto, imagen, etc.)
  • Padding: Espacio interno entre el contenido y el borde
  • Border: El borde del elemento
  • Margin: Espacio externo entre este elemento y otros

Content - Contenido

.box {
    width: 300px;
    height: 200px;
}

Padding - Espacio Interno

/* Padding uniforme */
.box {
    padding: 20px;
}

/* Padding vertical y horizontal */
.box {
    padding: 10px 20px;
}

/* Cada lado individual */
.box {
    padding-top: 10px;
    padding-right: 20px;
    padding-bottom: 10px;
    padding-left: 20px;
}

/* Notación abreviada (top, right, bottom, left) */
.box {
    padding: 10px 20px 10px 20px;
}

Border - Borde

/* Border completo */
.box {
    border: 2px solid black;
}

/* Border individual */
.box {
    border-width: 2px;
    border-style: solid;
    border-color: black;
}

/* Bordes específicos */
.box {
    border-top: 1px solid red;
    border-bottom: 2px dashed blue;
}

/* Border radius */
.box {
    border-radius: 10px;
}

Margin - Espacio Externo

/* Margin uniforme */
.box {
    margin: 20px;
}

/* Centrar horizontalmente */
.box {
    margin: 0 auto;
}

/* Margin negativo */
.box {
    margin-top: -10px;
}

/* Auto margins para centrar */
.container {
    width: 800px;
    margin-left: auto;
    margin-right: auto;
}

Box Sizing

Diferencia entre content-box y border-box:

  • content-box (default): width/height solo incluye el contenido
  • border-box (recomendado): width/height incluye padding y border
/* Default - content-box */
.box1 {
    box-sizing: content-box;
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    /* Ancho total: 300 + 40 + 10 = 350px */
}

/* Recomendado - border-box */
.box2 {
    box-sizing: border-box;
    width: 300px;
    padding: 20px;
    border: 5px solid black;
    /* Ancho total: 300px (incluye todo) */
}

/* Aplicar a todos los elementos */
* {
    box-sizing: border-box;
}

Display

/* Block - Ocupa todo el ancho disponible */
div {
    display: block;
}

/* Inline - Solo ocupa el espacio necesario */
span {
    display: inline;
}

/* Inline-block - Como inline pero acepta width/height */
.button {
    display: inline-block;
    width: 200px;
}

/* None - Oculta el elemento */
.hidden {
    display: none;
}
