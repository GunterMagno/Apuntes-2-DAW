CSS Grid - Layout Bidimensional

¿Qué es CSS Grid?

CSS Grid es un sistema de layout bidimensional que permite crear diseños complejos con filas y columnas.

.container {
    display: grid;
    grid-template-columns: 1fr 2fr 1fr;
    grid-template-rows: auto 1fr auto;
    gap: 20px;
}
