Mixins en Sass

¿Qué son los Mixins?

Los mixins son bloques de código reutilizables que pueden aceptar parámetros y generar CSS dinámico.

@mixin flex-center {
    display: flex;
    justify-content: center;
    align-items: center;
}

.container {
    @include flex-center;
}

Mixins Básicos

@mixin reset-list {
    margin: 0;
    padding: 0;
    list-style: none;
}

ul {
    @include reset-list;
}

Mixins con Parámetros

@mixin button($bg-color, $text-color: white) {
    background: $bg-color;
    color: $text-color;
    padding: 10px 20px;
    border-radius: 5px;
    border: none;
}

.btn-primary {
    @include button(#2965f1);
}

.btn-success {
    @include button(#28a745, #000);
}

@content en Mixins

@mixin responsive($breakpoint) {
    @if $breakpoint == mobile {
        @media (max-width: 768px) { @content; }
    } @else if $breakpoint == tablet {
        @media (max-width: 1024px) { @content; }
    }
}

.sidebar {
    width: 300px;
    
    @include responsive(mobile) {
        width: 100%;
    }
}
