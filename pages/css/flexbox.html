Flexbox - Sistema de Layout Unidimensional

¿Qué es Flexbox?

Flexbox es un sistema de layout unidimensional (filas o columnas) que facilita la distribución y alineación de elementos.

.container {
    display: flex;
}

Propiedades del Contenedor

.flex-container {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
    justify-content: center;
    align-items: center;
    gap: 20px;
}

Propiedades de los Items

.flex-item {
    flex-grow: 1;
    flex-shrink: 1;
    flex-basis: 200px;
    /* Shorthand */
    flex: 1 1 200px;
    
    align-self: flex-start;
    order: 2;
}
