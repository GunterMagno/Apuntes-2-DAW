Variables en Sass

¿Qué son las Variables?

Las variables en Sass permiten almacenar valores que se pueden reutilizar en todo el código CSS.

$primary-color: #2965f1;
$font-stack: 'Inter', sans-serif;
$spacing: 20px;

body {
    font-family: $font-stack;
    color: $primary-color;
    margin: $spacing;
}

Declaración de Variables

// Variables globales
$primary-color: #2965f1;
$secondary-color: #cd6799;
$dark-bg: #1a1a1a;

// Variables locales
.card {
    $card-padding: 20px;
    padding: $card-padding;
}

Scope de Variables

$global-var: #fff;

.container {
    $local-var: 10px; // Solo disponible en .container
    margin: $local-var;
    color: $global-var; // Acceso a variable global
}

// $local-var no existe aquí
.otro {
    color: $global-var; // ✓
    // margin: $local-var; // ✗ Error
}

Interpolación

$name: "header";
$side: "top";

.#{$name} {
    margin-#{$side}: 20px;
}

// Resultado:
// .header {
//     margin-top: 20px;
// }
