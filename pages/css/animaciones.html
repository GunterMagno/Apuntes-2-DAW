Animaciones y Transiciones CSS

Transitions

    background: blue;
    transition: background 0.3s ease;
}

.button:hover {

}

Animations

@keyframes slideIn {
    from {
        transform: translateX(-100%);
    }
    to {
        transform: translateX(0);
    }
}

.element {
    animation: slideIn 0.5s ease;
}
