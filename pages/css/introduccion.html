Introducción a CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación de documentos HTML. Mientras que HTML define la estructura y el contenido, CSS define cómo se ve ese contenido.

Ventajas de CSS

  • Separación de contenido y presentación: Facilita el mantenimiento
  • Reutilización: Un mismo CSS puede aplicarse a múltiples páginas
  • Control preciso: Control detallado sobre la apariencia
  • Rendimiento: Reduce la duplicación de código

¿Cómo funciona CSS?

CSS funciona mediante reglas que asocian selectores con declaraciones de estilo:

selector {
    propiedad: valor;
    propiedad: valor;
}

El proceso de renderizado

  1. El navegador carga el HTML
  2. Parsea el HTML y construye el DOM (Document Object Model)
  3. Carga y parsea el CSS
  4. Aplica los estilos al DOM según especificidad y cascada
  5. Renderiza la página con los estilos aplicados

Formas de aplicar CSS

1. CSS Externo (Recomendado)

Archivo .css separado vinculado desde el HTML:

<head>
    <link rel="stylesheet" href="styles.css">
</head>

2. CSS Interno

Dentro de la etiqueta <style> en el HTML:

<head>
    <style>
        body {
            background-color: #f0f0f0;
        }
    </style>
</head>

3. CSS Inline

Directamente en el atributo style del elemento:

<p style="color: blue; font-size: 18px;">Texto</p>

Importante

El CSS inline tiene la mayor especificidad y debe usarse con moderación. Es preferible usar CSS externo para mantener la separación de preocupaciones.

Sintaxis Básica

Anatomía de una regla CSS

/* Comentario en CSS */
selector {
    propiedad: valor;
    otra-propiedad: otro-valor;
}
  • Selector: Indica qué elementos se estilizan
  • Declaración: Par propiedad-valor
  • Propiedad: Aspecto que se quiere cambiar
  • Valor: Nuevo valor para la propiedad

Ejemplo completo

/* Estiliza todos los párrafos */
p {
    color: #333;
    font-size: 16px;
    line-height: 1.6;
    margin-bottom: 20px;
}

Selectores Básicos

Selector de tipo

Selecciona todos los elementos de un tipo:

p {
    color: blue;
}

h1 {
    font-size: 32px;
}

Selector de clase

Selecciona elementos con una clase específica:

.highlight {
    background-color: yellow;
}

.btn-primary {
    background: blue;
    color: white;
}

Selector de ID

Selecciona un elemento único por su ID:

#header {
    height: 100px;
}

#main-content {
    max-width: 1200px;
}

Selector universal

Selecciona todos los elementos:

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

Selectores combinados

/* Descendiente */
article p {
    line-height: 1.8;
}

/* Hijo directo */
ul > li {
    list-style: none;
}

/* Hermano adyacente */
h1 + p {
    font-size: 18px;
}

/* Múltiples selectores */
h1, h2, h3 {
    font-family: 'Arial', sans-serif;
}

Propiedades CSS Comunes

Colores y Fondos

.ejemplo {
    /* Color de texto */
    color: #333;
    color: rgb(51, 51, 51);
    color: rgba(51, 51, 51, 0.8);
    
    /* Color de fondo */
    background-color: #f0f0f0;
    
    /* Imagen de fondo */
    background-image: url('imagen.jpg');
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
}

Texto

.texto {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    font-weight: bold;
    font-style: italic;
    
    text-align: center;
    text-decoration: underline;
    text-transform: uppercase;
    
    line-height: 1.6;
    letter-spacing: 2px;
}

Box Model

.caja {
    /* Ancho y alto */
    width: 300px;
    height: 200px;
    
    /* Padding (espacio interno) */
    padding: 20px;
    padding-top: 10px;
    
    /* Margin (espacio externo) */
    margin: 20px;
    margin-bottom: 30px;
    
    /* Border */
    border: 2px solid #333;
    border-radius: 8px;
}

Display y Posicionamiento

.elemento {
    /* Display */
    display: block;
    display: inline;
    display: inline-block;
    display: flex;
    display: grid;
    display: none;
    
    /* Position */
    position: static;
    position: relative;
    position: absolute;
    position: fixed;
    position: sticky;
    
    /* Coordenadas */
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    
    /* Z-index */
    z-index: 100;
}
