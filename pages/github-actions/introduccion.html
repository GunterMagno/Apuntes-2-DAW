1. Introducción a GitHub Actions

Automatización en GitHub

¿Qué es GitHub Actions?

GitHub Actions es una plataforma de integración y entrega continua (CI/CD) que permite automatizar tu pipeline de desarrollo.

Características:
  • Integrado nativamente en GitHub
  • Ejecuta workflows automáticos
  • Múltiples runners (Linux, Windows, macOS)
  • Marketplace de actions reutilizables

CI/CD

Continuous Integration (CI): Integrar cambios de código frecuentemente.

Continuous Deployment (CD): Desplegar automáticamente a producción.

Ventajas

  • Automatización de tests
  • Despliegue automático
  • Detección temprana de errores
  • Feedback rápido

Conceptos Básicos

  • Workflow: Proceso automatizado
  • Job: Conjunto de steps
  • Step: Tarea individual
  • Action: Comando reutilizable
  • Runner: Servidor que ejecuta workflows
