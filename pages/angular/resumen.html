Comandos CLI, decoradores y directivas

Angular CLI - Comandos Principales

Comando Flags Comunes Descripción
ng new nombre-proyecto Crea nuevo proyecto Angular
ng serve -o (abrir navegador), --port 4200 Inicia servidor de desarrollo
ng build --prod / --configuration production Construye aplicación para producción
ng test - Ejecuta tests unitarios (Karma)
ng e2e - Ejecuta tests end-to-end
ng generate Ver tabla siguiente Genera elementos (componentes, servicios, etc.)
ng update @angular/cli @angular/core Actualiza Angular y dependencias
ng add paquete Añade dependencia al proyecto

ng generate - Crear Elementos

Comando Alias Descripción
ng generate component nombre ng g c nombre Crea componente
ng generate service nombre ng g s nombre Crea servicio
ng generate module nombre ng g m nombre Crea módulo
ng generate directive nombre ng g d nombre Crea directiva
ng generate pipe nombre ng g p nombre Crea pipe
ng generate guard nombre ng g g nombre Crea guard (protección de rutas)
ng generate interface nombre ng g i nombre Crea interfaz TypeScript
ng generate class nombre ng g cl nombre Crea clase TypeScript

Decoradores Principales

Decorador Uso Descripción
@Component Clase de componente Define un componente Angular
@NgModule Clase de módulo Define un módulo Angular
@Injectable Clase de servicio Marca clase como inyectable
@Directive Clase de directiva Define una directiva
@Pipe Clase de pipe Define un pipe personalizado
@Input Propiedad de componente Recibe datos del padre
@Output Propiedad EventEmitter Emite eventos al padre
@ViewChild Propiedad Accede a hijo en template
@HostListener Método Escucha eventos del host

Directivas Estructurales

Directiva Sintaxis Descripción
*ngIf <div *ngIf="condicion"> Renderiza si condición es verdadera
*ngIf else <div *ngIf="cond; else template"> Renderiza con alternativa
*ngFor <li *ngFor="let item of items"> Itera sobre array
*ngFor index *ngFor="let item of items; let i = index" Accede al índice
*ngSwitch <div [ngSwitch]="valor"> Switch/case en template
*ngSwitchCase <p *ngSwitchCase="'valor'"> Caso del switch
*ngSwitchDefault <p *ngSwitchDefault> Caso por defecto

Directivas de Atributo

Directiva Sintaxis Descripción
[ngClass] [ngClass]="{'clase': condicion}" Añade clases CSS condicionalmente
[ngStyle] [ngStyle]="{'color': variable}" Aplica estilos inline dinámicos
[ngModel] [(ngModel)]="propiedad" Two-way data binding (requiere FormsModule)

Data Binding

Tipo Sintaxis Descripción
Interpolación {{ variable }} Muestra datos en template
Property Binding [propiedad]="valor" Vincula propiedad de elemento
Event Binding (evento)="metodo()" Escucha eventos
Two-way Binding [(ngModel)]="propiedad" Sincronización bidireccional
Template Reference #nombre Referencia a elemento en template

Routing - Router

Concepto Sintaxis/Código Descripción
RouterModule.forRoot() RouterModule.forRoot(routes) Configuración de rutas raíz
Routes { path: 'home', component: HomeComponent } Define ruta
routerLink <a routerLink="/home"> Navegación declarativa
routerLinkActive routerLinkActive="active" Clase CSS en ruta activa
router-outlet <router-outlet></router-outlet> Punto de renderizado de rutas
Router.navigate() this.router.navigate(['/home']) Navegación programática
ActivatedRoute route.params.subscribe(p => p.id) Accede a parámetros de ruta
Path param { path: 'user/:id', ... } Parámetro en ruta

Servicios y HTTP

Concepto Ejemplo Descripción
HttpClient constructor(private http: HttpClient) Cliente HTTP de Angular
http.get() this.http.get<T>(url) Petición GET
http.post() this.http.post(url, body) Petición POST
http.put() this.http.put(url, body) Petición PUT
http.delete() this.http.delete(url) Petición DELETE
Observable .subscribe(data => {}) Suscripción a Observable (RxJS)
pipe() observable.pipe(map(), filter()) Operadores RxJS

Pipes Comunes

Pipe Sintaxis Descripción
date {{ fecha | date:'dd/MM/yyyy' }} Formatea fechas
uppercase {{ texto | uppercase }} Convierte a mayúsculas
lowercase {{ texto | lowercase }} Convierte a minúsculas
currency {{ precio | currency:'EUR' }} Formatea moneda
percent {{ valor | percent }} Formatea porcentaje
json {{ objeto | json }} Muestra objeto como JSON
slice {{ array | slice:0:5 }} Corta array o string
async {{ observable$ | async }} Suscribe a Observable automáticamente

Formularios

Tipo Módulo Descripción
Template-Driven FormsModule Formularios basados en template con ngModel
Reactive Forms ReactiveFormsModule Formularios reactivos (FormGroup, FormControl)
FormControl new FormControl('valor inicial') Control individual de formulario
FormGroup new FormGroup({campo: new FormControl()}) Grupo de controles
FormBuilder fb.group({campo: ['valor']}) Constructor de formularios
Validators Validators.required, Validators.email Validadores integrados
