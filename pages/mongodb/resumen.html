Resumen MongoDB

Comandos y operaciones CRUD esenciales

Comandos Básicos de MongoDB Shell

Comando Parámetros Descripción
mongosh - Inicia MongoDB Shell
show dbs - Muestra todas las bases de datos
use nombreDB Cambia/crea base de datos
db - Muestra base de datos actual
show collections - Muestra colecciones de la BD actual
db.dropDatabase() - Elimina base de datos actual

CRUD - Create (Insertar)

Método Sintaxis Descripción
insertOne() db.coleccion.insertOne({...}) Inserta un documento
insertMany() db.coleccion.insertMany([{...}, {...}]) Inserta múltiples documentos

CRUD - Read (Consultar)

Método Sintaxis Descripción
find() db.coleccion.find() Encuentra todos los documentos
find() db.coleccion.find({campo: valor}) Encuentra con filtro
findOne() db.coleccion.findOne({...}) Encuentra un solo documento
.limit() db.coleccion.find().limit(5) Limita resultados
.sort() db.coleccion.find().sort({campo: 1}) Ordena (1: asc, -1: desc)
.skip() db.coleccion.find().skip(10) Salta documentos
.count() db.coleccion.find().count() Cuenta documentos

CRUD - Update (Actualizar)

Método Sintaxis Descripción
updateOne() db.coleccion.updateOne({filtro}, {$set: {...}}) Actualiza un documento
updateMany() db.coleccion.updateMany({filtro}, {$set: {...}}) Actualiza múltiples documentos
replaceOne() db.coleccion.replaceOne({filtro}, {...}) Reemplaza documento completo

CRUD - Delete (Eliminar)

Método Sintaxis Descripción
deleteOne() db.coleccion.deleteOne({filtro}) Elimina un documento
deleteMany() db.coleccion.deleteMany({filtro}) Elimina múltiples documentos
drop() db.coleccion.drop() Elimina colección completa

Operadores de Consulta

Operador Sintaxis Descripción
$eq {campo: {$eq: valor}} Igual a
$ne {campo: {$ne: valor}} No igual a
$gt {campo: {$gt: valor}} Mayor que
$gte {campo: {$gte: valor}} Mayor o igual que
$lt {campo: {$lt: valor}} Menor que
$lte {campo: {$lte: valor}} Menor o igual que
$in {campo: {$in: [v1, v2]}} Está en array
$nin {campo: {$nin: [v1, v2]}} No está en array
$and {$and: [{...}, {...}]} AND lógico
$or {$or: [{...}, {...}]} OR lógico
$not {campo: {$not: {...}}} NOT lógico
$exists {campo: {$exists: true}} Verifica si campo existe

Operadores de Actualización

Operador Sintaxis Descripción
$set {$set: {campo: valor}} Establece valor de campo
$unset {$unset: {campo: ""}} Elimina campo del documento
$inc {$inc: {campo: 1}} Incrementa valor numérico
$mul {$mul: {campo: 2}} Multiplica valor
$rename {$rename: {viejo: "nuevo"}} Renombra campo
$push {$push: {array: valor}} Añade elemento a array
$pull {$pull: {array: valor}} Elimina elemento de array
