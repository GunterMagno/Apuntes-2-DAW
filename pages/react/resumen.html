Resumen React

Hooks, componentes y comandos esenciales

Comandos CLI

Comando Descripción
npx create-react-app nombre-app Crea nueva aplicación React
npm start Inicia servidor de desarrollo
npm run build Crea build de producción
npm test Ejecuta tests
npm install react-router-dom Instala React Router

Hooks Principales

Hook Sintaxis Descripción
useState const [state, setState] = useState(valorInicial) Maneja estado en componente funcional
useEffect useEffect(() => { }, [deps]) Efectos secundarios y ciclo de vida
useContext const value = useContext(MyContext) Consume contexto React
useRef const ref = useRef(valorInicial) Referencia mutable que persiste renders
useReducer const [state, dispatch] = useReducer(reducer, initialState) Alternativa a useState para lógica compleja
useMemo const memoizedValue = useMemo(() => computeExpensive(a), [a]) Memoriza cálculos costosos
useCallback const memoizedFn = useCallback(() => {}, [deps]) Memoriza funciones
useNavigate const navigate = useNavigate() Navegación programática (React Router v6)
useParams const { id } = useParams() Obtiene parámetros de URL (React Router)

Sintaxis JSX Esencial

Concepto Sintaxis Descripción
Expresiones {variable} o {expresion} Inserta JavaScript en JSX
Atributos className, htmlFor Usa camelCase para atributos HTML
Eventos onClick={handleClick} Manejo de eventos en camelCase
Condicional {condicion && } Renderizado condicional
Ternario {condicion ? : } Condicional con alternativa
Map {items.map(item => )} Renderiza listas (requiere key)
Fragmentos <>...</> o <React.Fragment> Agrupa elementos sin nodo DOM

Props y Componentes

Concepto Ejemplo Descripción
Componente Funcional function MiComponente(props) { return ... } Componente como función
Props <Componente prop1="valor" prop2={var} /> Pasa datos al componente
Destructuring function Comp({ prop1, prop2 }) {} Extrae props directamente
Children props.children Contenido entre tags del componente
Default Props Comp.defaultProps = { prop: 'default' } Valores por defecto de props

React Router (v6)

Componente/Hook Sintaxis Descripción
BrowserRouter <BrowserRouter><App /></BrowserRouter> Envuelve la aplicación
Routes <Routes>...</Routes> Contenedor de rutas
Route <Route path="/about" element={<About />} /> Define una ruta
Link <Link to="/about">About</Link> Navegación declarativa
NavLink <NavLink to="/home" className={({isActive}) => ...} /> Link con estado activo
useNavigate navigate('/ruta') Navegación programática
useParams const { id } = useParams() Obtiene parámetros de URL
useLocation const location = useLocation() Obtiene información de ubicación

Fetch & Axios

Método Sintaxis Descripción
fetch GET fetch(url).then(res => res.json()) Petición GET con Fetch API
fetch POST fetch(url, {method: 'POST', body: JSON.stringify(data)}) Petición POST con Fetch
axios.get axios.get(url).then(res => res.data) GET con Axios
axios.post axios.post(url, data) POST con Axios
async/await const res = await axios.get(url) Uso con async/await

Conceptos Clave

Concepto Descripción
Virtual DOM Representación en memoria del DOM real para optimizar renders
Estado (State) Datos internos del componente que pueden cambiar
Props Datos que se pasan de padre a hijo (inmutables)
One-way Data Flow Los datos fluyen de padres a hijos
Lifting State Up Mover estado al componente padre común
Controlled Components Componentes de formulario controlados por React state
