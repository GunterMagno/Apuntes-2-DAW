Resumen Rápido: MongoDB

Referencia rápida de operaciones CRUD, agregaciones y comandos

Comandos de MongoDB Shell

Comando Descripción
mongosh Inicia el shell de MongoDB
show dbs Muestra todas las bases de datos
use nombreDB Cambia o crea una base de datos
show collections Muestra todas las colecciones de la BD actual
db.dropDatabase() Elimina la base de datos actual
db.createCollection('nombre') Crea una nueva colección
db.coleccion.drop() Elimina una colección

CREATE - Insertar documentos

Método Qué hace Ejemplo
insertOne(doc) Inserta un solo documento db.users.insertOne({name: "Juan", age: 25})
insertMany([docs]) Inserta múltiples documentos db.users.insertMany([{...}, {...}])

READ - Consultar documentos

Método Qué hace Ejemplo
find() Obtiene todos los documentos db.users.find()
find(query) Obtiene documentos que cumplan el query db.users.find({age: {$gt: 18}})
findOne(query) Obtiene el primer documento que coincida db.users.findOne({name: "Juan"})
find().limit(n) Limita resultados a n documentos db.users.find().limit(10)
find().skip(n) Salta los primeros n resultados db.users.find().skip(5)
find().sort({campo: 1}) Ordena resultados (1: ascendente, -1: descendente) db.users.find().sort({age: -1})
countDocuments() Cuenta documentos que coincidan db.users.countDocuments({age: {$gte: 18}})

UPDATE - Actualizar documentos

Método Qué hace Ejemplo
updateOne(query, update) Actualiza el primer documento que coincida db.users.updateOne({name: "Juan"}, {$set: {age: 26}})
updateMany(query, update) Actualiza todos los documentos que coincidan db.users.updateMany({city: "Madrid"}, {$set: {country: "Spain"}})
replaceOne(query, doc) Reemplaza completamente un documento db.users.replaceOne({_id: id}, {name: "Pedro", age: 30})

DELETE - Eliminar documentos

Método Qué hace Ejemplo
deleteOne(query) Elimina el primer documento que coincida db.users.deleteOne({name: "Juan"})
deleteMany(query) Elimina todos los documentos que coincidan db.users.deleteMany({age: {$lt: 18}})

Operadores de consulta más usados

Operador Significado Ejemplo
$eq Igual a {age: {$eq: 25}}
$ne No igual a {status: {$ne: "inactive"}}
$gt Mayor que {age: {$gt: 18}}
$gte Mayor o igual que {age: {$gte: 18}}
$lt Menor que {price: {$lt: 100}}
$lte Menor o igual que {price: {$lte: 50}}
$in Valor está en array {status: {$in: ["active", "pending"]}}
$nin Valor NO está en array {color: {$nin: ["red", "blue"]}}
$and AND lógico {$and: [{age: {$gt: 18}}, {city: "Madrid"}]}
$or OR lógico {$or: [{age: {$lt: 18}}, {status: "student"}]}
$not NOT lógico {age: {$not: {$gt: 18}}}
$exists Campo existe o no {email: {$exists: true}}
$regex Expresión regular {name: {$regex: /^J/i}}

Operadores de actualización

Operador Qué hace Ejemplo
$set Establece valor de un campo {$set: {age: 26}}
$unset Elimina un campo {$unset: {temporaryField: ""}}
$inc Incrementa valor numérico {$inc: {views: 1}}
$mul Multiplica valor numérico {$mul: {price: 1.1}}
$rename Renombra un campo {$rename: {oldName: "newName"}}
$push Añade elemento a array {$push: {tags: "new"}}
$pull Elimina elemento de array {$pull: {tags: "old"}}
$addToSet Añade a array si no existe {$addToSet: {tags: "unique"}}

Índices - Mejorar rendimiento

Comando Qué hace Ejemplo
createIndex() Crea un índice en un campo db.users.createIndex({email: 1})
createIndex() unique Crea índice único (no duplicados) db.users.createIndex({email: 1}, {unique: true})
getIndexes() Lista todos los índices db.users.getIndexes()
dropIndex() Elimina un índice db.users.dropIndex("email_1")

Pipeline de agregación

Para análisis y transformaciones complejas de datos.

Stage Qué hace Ejemplo
$match Filtra documentos (como find) {$match: {age: {$gte: 18}}}
$group Agrupa documentos y calcula {$group: {_id: "$city", total: {$sum: 1}}}
$sort Ordena documentos {$sort: {age: -1}}
$project Selecciona/transforma campos {$project: {name: 1, age: 1, _id: 0}}
$limit Limita número de documentos {$limit: 10}
$skip Salta documentos {$skip: 5}
$lookup Join con otra colección {$lookup: {from: "orders", localField: "_id", ...}}
$unwind Descompone array en documentos {$unwind: "$tags"}

Mongoose - ODM para Node.js

Concepto/Método Descripción Ejemplo
mongoose.connect() Conecta a MongoDB mongoose.connect('mongodb://localhost/mydb')
Schema Define estructura del documento new Schema({ name: String, age: Number })
Model Crea modelo desde schema mongoose.model('User', userSchema)
save() Guarda documento en BD await user.save()
Model.create() Crea y guarda documento await User.create({name: "Juan"})
Model.find() Encuentra documentos await User.find({age: {$gt: 18}})
Model.findById() Encuentra por ID await User.findById(id)
Model.findByIdAndUpdate() Actualiza por ID await User.findByIdAndUpdate(id, {age: 26})
Model.findByIdAndDelete() Elimina por ID await User.findByIdAndDelete(id)

Consejo importante

MongoDB es flexible: no necesitas definir esquemas rígidos. Cada documento puede tener campos diferentes.

Diseña pensando en consultas: a diferencia de SQL, en MongoDB a veces duplicas datos para optimizar lecturas.

Usa índices: son cruciales para rendimiento en colecciones grandes. Indexa campos que usas frecuentemente en queries.
